- 202255161 이민지 역할: 네이버책 사이트에서 검색어에 대한 책목록을 크롤링하는 크롤러 코딩 후 github에 업로드

- 202255169 이지훈 역할: 코딩한 크롤러가 사이트에서 검색어를 입력하면 책의 목록들을 가격별로 가져오는 데이터를 크롤링하는 크롤러를 코딩 후 github에 업로드
